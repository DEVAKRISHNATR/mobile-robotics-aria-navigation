## Overview
This project implements autonomous mobile robot navigation using the ARIA simulator.
The system focuses on odometry-based waypoint navigation, obstacle avoidance, and experimental performance evaluation.

## Features
- Odometry-based navigation
- Waypoint control
- Obstacle avoidance
- Trajectory and distance analysis
- Multiple experimental runs

## Tools & Technologies
- ARIA Simulator
- Mobile Robot (Pioneer)
- Python / C++
- Odometry & control algorithms

## Project Type
Academic project developed as part of an MSc in Robotics & Automation.

## Results
Experimental evaluation includes trajectory plots, distance metrics, and time analysis across multiple runs.
